Tool Guide - Tour Builder Pro

Side Panel (Identity and Floor Plan)

  • Logo and Global Title: Define the branding of your virtual tour. It will always be displayed in the lower-left corner. You can change the text size and color.
  • Pointer Color (Hotspots): Allows you to customize the color of the navigation arrows so they match your corporate identity.
  • Floor Plan / Minimap: Upload a 2D image of the architectural floor plan. It enables an interactive minimap where you can place the photos.
  • 360° Scenes: Upload your equirectangular panoramic photos (2:1 ratio). The system automatically optimizes them in 4K format for better performance.

Editing Panel (Thumbnails)

  • Add Subtitle and Information: Select a scene in the list and add a subtitle and a descriptive text block ("i" icon).
  • Delete Scenes: Use the trash icon next to each scene. Note: When deleting, make sure to update any arrows that pointed there.
  • Save and Export: You can save a single HTML file (ideal for quickly sending via WhatsApp or email) or a complete ZIP file (better for uploading to professional web servers).

Minimap Tools (Location Panel)

🖼️ Gallery Button

Displays a bottom bar with all thumbnail images, allowing the end user to quickly jump from one room to another.

🔗 Add Arrow (Hotspot)

Connects the images to each other. Click, choose the destination, optionally rename the name assigned to the arrow (Example: Living Room), and use the mouse to mark on the screen where the transition arrow will go. Optionally, you can click Advanced Settings to enable the option to set the entry view when clicking the arrow.

ℹ️ Info Hotspot

Place information markers (green icon) on objects (a window, an appliance) to describe important qualities or features. Add text and then use the mouse to mark on the screen where the information point will go

📍 Set Starting Point and Radar

Adjust the initial view of the photo when entering it, and calibrate where the red arrow points on the minimap floor plan.

🔄 Transition List

View, manage, and delete the navigation arrows created in the current scene.

🔒 Lock Map

Locks the points (markers) on the minimap to avoid moving them by mistake while clicking as you navigate.
